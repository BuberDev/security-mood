# Security Mood — Professional Preparedness Platform

Security Mood is a high-intent Amazon affiliate platform built with Next.js, designed to convert traffic from Pinterest through authoritative "Operational Protocol" guides.

## 🛡️ Brand Vision
We help modern homeowners and families build safety into their daily lives. From home fortification to digital privacy, Security Mood curates battle-tested gear and provides step-by-step instructions for a more resilient lifestyle.

## 🚀 Key Features
- **Operational Protocols**: High-converting guides that transform complex security topics into actionable steps.
- **Pinterest-First Design**: Vertical imagery and emotional "hooks" tailored for Pinterest traffic.
- **Amazon Associates Optimized**: Deep-linked product collections and "Field Performance" proofs for maximum trust and click-through rates.
- **Fully Responsive**: Premium, mobile-first experience using Framer Motion for smooth transitions.

## 🛠️ Tech Stack
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS 4.0
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Fonts**: Inter & Playfair Display

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Project Structure
- `/app`: Next.js App Router and page layouts.
- `/lib/site-data.ts`: Central source of truth for all products, categories, and articles.
- `/lib/product-page-content.ts`: Narrative layer and ritual instructions for each category.
- `/components`: Reusable UI components including the "Security Field Collection" cards.
